# nextjs-example-app

## Getting Started

```sh
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

```sh
# Install dependencies
npm install

# Lint code
npm run lint 

# Run unit tests
npm test

# Build production dist
npm run build
```
